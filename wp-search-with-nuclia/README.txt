== Description ==

Improve search on your site.

= Features =
* Push your content to NucliaDB for indexing
* Nuclia searchbox widget
* Nuclia searchbox shortcode

This plugin requires a Nuclia account. You can sign up for free at [Nuclia](https://nuclia.com).

= Links =
* [Nuclia](https://nuclia.com)

== Installation ==


From your WordPress dashboard:

1. **Visit** Plugins > Add New
2. **Search** for "WP Search With Nuclia"
3. **Activate** WP Search With Nuclia from your Plugins page
4. **Click** on the new menu item "Nuclia Search" and enter your API keys


= What are the minimum requirements? =

* Requires WordPress 5.6+
* PHP version 7.2 or greater (PHP 7.4 is recommended)
* MySQL version 5.0 or greater (MySQL 5.6 or greater is recommended)
* cURL PHP extension
* mbstring PHP extension
* OpenSSL greater than 1.0.1
